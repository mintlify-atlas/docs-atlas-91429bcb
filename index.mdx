C^2M-Engine (C2ME) is a Fabric mod that accelerates Minecraft chunk generation, I/O, and loading by running work across multiple CPU cores in parallel. Whether you're running a large server or a single-player world, C2ME helps the game keep up with your view distance demands without sacrificing vanilla behavior.

## What C2ME does

C2ME targets the three most expensive parts of Minecraft's chunk pipeline:

- **Chunk generation** — world terrain, biomes, structures, and features are generated in parallel across multiple threads instead of one at a time
- **Chunk I/O** — reading and writing chunk data to disk is handled asynchronously, keeping the main server thread free
- **Chunk loading** — the chunk system scheduler is rewritten to maximize throughput and reduce latency

## Recommended companions

For best performance, use C2ME alongside:

- **[Lithium](https://github.com/CaffeineMC/lithium-fabric)** — general-purpose server optimization mod
- **[Starlight](https://github.com/Spottedleaf/Starlight)** — lighting engine rewrite

<Note>
  Always back up your worlds before installing or updating C2ME. World data is safe under normal operation, but good backup hygiene is always recommended when using performance mods.
</Note>
